My Recent Projects

Campus Map

Campus Map is an app designed for Penn State Students by a Penn State Student.

It allows a user to locate any building on the Main PSU Campus. When a building is selected its pin is displayed on the map. Tapping this pin displays the name, year-constructed, and image of the building. A route to this building from their current location or another set location can be generated. The user can pin any location on the map and swipe between standard, hybrid, or satellite map views.
